How it works

nRF Web Tool works by combining Web Serial, Web Serial Polyfill (for Android), and DFU packages which contain firmware binaries and metadata. The tool detects devices with the Adafruit nRF bootloader and automatically installs the appropriate firmware.

Web Serial is available in Google Chrome and Microsoft Edge browsers. Android support is available via the Web Serial Polyfill, which implements Web Serial on top of WebUSB.

Adding nRF Web Tool to your website

To add this to your own website, you need to include the nRF Web Tool JavaScript files on your website and add the button HTML.

Step 1:

Load nRF Web Tool JavaScript on your website by adding the following HTML snippet.

<script src="https://opendisplay.org/nrf_web_tools/jszip.min.js"></script>

<!-- Web Serial Polyfill for Android support (optional, must be before nrf_web_tools.js) -->
<script type="module">
  import { serial } from 'https://opendisplay.org/nrf_web_tools/web-serial-polyfill.js';
  const isAndroid = /Android/i.test(navigator.userAgent);
  if (isAndroid && navigator.usb) {
    navigator.serial = serial;
  } else if (!navigator.serial) {
    navigator.serial = serial;
  }
</script>

<script src="https://opendisplay.org/nrf_web_tools/nrf_web_tools.js"></script>

Step 2:

Find a place on your page where you want the button to appear and include the following bit of HTML. Update the zipPath to point at your DFU package.

<button id="install-btn">Install Firmware</button>

<script>
  window.addEventListener('load', () => {
    const installer = new NrfWebTools(
      document.getElementById('install-btn'),
      'https://example.com/firmware.zip'  // URL to your DFU package
    );
  });
</script>
Note: nRF Web Tool requires that your website is served over https:// to work. This is a Web Serial security requirement. For local development, localhost is also allowed.

API Reference

Constructor

new NrfWebTools(buttonElement, zipPath)
  • buttonElement - The button element to attach the installer to
  • zipPath - URL string or File object pointing to the DFU package ZIP file

Methods

  • startInstall() - Manually trigger the installation (usually called automatically on button click)
  • closeModal() - Close the progress modal
